Sudoku

To enter a digit in a cell:

  1. Touch the cell (becomes blue)
  2. Touch the digit in the bottom row
To delete a cell, press the "X" button instead of a digit.

After input:
  • The edited cell is protected from the solver (becomes orange)
  • The unprotected cells are deleted
  • The solver is reset
To solve the sudoku, press the "SOLVE" button.

Icon Function
Keep Screen
(ON/OFF) (highlighted if ON)
Reset Delete all cells (unprotected and protected)
Reset U Delete all unprotected cells
Solve Solve the sudoku.
(becomes green if a solution is found, or red if impossible to solve)
Press it again to display the next solution.
